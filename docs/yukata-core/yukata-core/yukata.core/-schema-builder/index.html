Represents a builder class to construct Schemas.

Example

val schema = createSchema {
    // Register resolvers
    resolver(UserResolver)
    resolver(LoginResolver)

    // Register types
    objectType(SomeObjectType)

    // Register scalars
    scalar(SomeScalar)

    // ...
}
